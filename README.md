# 🏠 お部屋レイアウトツール ver.2.3

ブラウザだけで動作する、インタラクティブな部屋のレイアウト作成ツールです。  
家具の配置・編集から間取りのフリーフォーム作成まで、直感的な操作で理想の部屋レイアウトを設計できます。

---

## 📸 概要

- **インストール不要** — `index.html` をブラウザで開くだけで動作します
- **日本の住宅規格に対応** — 京間・中京間・江戸間・団地間の畳数プリセットを収録
- **フリーフォーム描画** — L字形など任意形状の家具・間取りを自由に作成
- **画像保存対応** — レイアウト図面をPNGとしてダウンロード可能

---

## ✨ 主な機能

### 部屋の設定
| 機能 | 説明 |
|------|------|
| 畳数・間の種類から自動設定 | 京間 / 中京間 / 江戸間 / 団地間 × 各畳数のプリセットサイズを自動入力 |
| 縦長 / 横長 / 正方形 | 部屋の向きを選択（正方形は対応畳数のみ有効） |
| 幅・奥行き入力 | cm単位で任意サイズを直接入力 |
| 大部屋の自動縮尺 | 画面に収まらない部屋を自動でスケールダウン表示 |
| 床のデザイン変更 | ホワイト / オーク / ウォルナット / メープル / チェリー / グレーの6種類 |

### 間取りのフリーフォーム作成
- 多角形モード（クリックで頂点を追加）またはフリーハンドモードで任意形状の間取りを描画
- 壁の色・床の塗りつぶし色・壁の太さを設定可能
- 幅・奥行き（cm）を指定して実寸にスケール
- 作成後も「再編集」で修正可能、「削除して通常に戻す」で矩形部屋に戻せる

### 家具の追加・編集
| 機能 | 説明 |
|------|------|
| 家具名 | 任意の名前を設定 |
| 幅・奥行き | cm単位でサイズ指定 |
| 面の色 | カラーピッカーで選択、リアルタイムプレビュー |
| 枠の色 | カラーピッカーで選択 |
| 文字色 | カラーピッカーで任意色に変更可能 |
| 不透明度 | スライダーで10〜100%を調整、リアルタイムプレビュー |
| 形状 | 四角形 / 円形 |

### フリーフォームで家具を作成
- 多角形モード・フリーハンドモードで任意形状の家具を作成（L字形・T字形など）
- 角の丸みをスライダーで調整
- 作成後に名前・サイズ・色・文字色を編集可能

### 家具の操作
| 操作 | 方法 |
|------|------|
| 移動 | ドラッグ＆ドロップ |
| 回転 | 上部の回転ハンドルをドラッグ（15°刻み） |
| レイヤー変更 | 下部のボタン（⤒ 最前面 / ↑ 前面 / ↓ 背面 / ⤓ 最背面） |
| 削除 | 右下のゴミ箱アイコンにドラッグ |
| 編集 | クリックして選択 → パネルで編集 → 「作成/更新」ボタン |

### その他
- **ズーム機能** — ＋/－ボタンで20〜300%の範囲でズーム（10%刻み）
- **タイトル設定** — 画像保存時にタイトルを表示
- **画像保存** — レイアウト全体をPNG形式でダウンロード

---

## 🚀 使い方

1. `index.html` をダウンロードする
2. ブラウザ（Chrome / Edge / Firefox 推奨）で開く
3. 左パネルで部屋のサイズを設定する
4. 家具を追加・配置してレイアウトを作成する
5. 「画像を保存」でPNGとして書き出す

> **Note**: インターネット接続が必要です（html2canvas ライブラリを CDN から読み込んでいます）
